Útmutató

ISO írás BalenaEtcher-rel és alap WM alkalmazások telepítése.

ISO írás BalenaEtcher-rel

  1. Töltsd le a BalenaEtcher alkalmazást: etcher.balena.io
  2. Indítsd el, válaszd a Flash from file opciót, és tallózd be az ISO-t.
  3. Válaszd ki a cél USB meghajtót.
  4. Kattints a Flash! gombra, várd meg az írást és ellenőrzést.
  5. Bootolj az elkészült USB meghajtóról.

Alap WM alkalmazások telepítése

Ajánlott alap alkalmazások egy minimalista WM környezethez:

  • Alacritty – gyors terminál emulátor
  • Thunar – könnyű fájlkezelő
  • MPV – médialejátszó
  • Firefox – webböngésző

Telepítés Arch Linuxon:

sudo pacman -S alacritty thunar mpv firefox
